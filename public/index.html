Tablas de Datos

Exploración del modelo dimensional completo

FACT

fact_delitos

Tabla de hechos principal. Registros de delitos cruzando las tres dimensiones.

  • FK fecha_id
  • FK ubicacion_id
  • FK sexo_id
  • INT cantidad
438,526 registros
DIM

dim_fecha

Dimensión de tiempo con año, mes, día y día de semana.

  • PK fecha_id
  • DATE fecha_hecho
  • INT anio / mes / dia
  • INT dia_semana
  • VARCHAR nombre_mes
8,552 registros
DIM

dim_ubicacion

Dimensión geográfica con jerarquía departamento → municipio.

  • PK ubicacion_id
  • INT cod_depto / cod_muni
  • VARCHAR departamento
  • VARCHAR municipio
  • VARCHAR zona
2,217 registros
DIM

dim_sexo

Catálogo de categorías de género registradas.

  • PK sexo_id
  • VARCHAR sexo
5 registros
